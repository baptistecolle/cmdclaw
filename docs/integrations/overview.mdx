---
title: "Integrations"
description: "All the services CmdClaw can connect to and what it can do with them."
---

CmdClaw connects to 13 services across productivity, engineering, sales, and communication. Each integration uses OAuth so you just click connect and authorize.

To connect an integration, go to **Settings > Integrations** in the app.

## Google

<AccordionGroup>
  <Accordion title="Gmail">
    Read, search, and send emails.

    - List emails
    - Read email
    - Get unread emails
    - Send email
  </Accordion>
  <Accordion title="Google Calendar">
    Manage your calendar and events.

    - List calendars
    - List events
    - Get today's events
    - Create, update, and delete events
  </Accordion>
  <Accordion title="Google Docs">
    Read and create documents.

    - List and search documents
    - Read document content
    - Create documents
    - Append to documents
  </Accordion>
  <Accordion title="Google Sheets">
    Read and write spreadsheet data.

    - List and read spreadsheets
    - Create spreadsheets
    - Append rows, update cells, clear data
    - Add sheets
  </Accordion>
  <Accordion title="Google Drive">
    Browse, search, and manage files.

    - List and search files
    - Download and upload files
    - Create folders
    - Delete files
  </Accordion>
</AccordionGroup>
